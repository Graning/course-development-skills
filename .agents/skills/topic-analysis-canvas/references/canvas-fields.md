# 课题分析画布字段参考

模板工作簿包含两个工作表：

- `课题分析画布`：空白模板。
- `示例`：以“如何做新员工带教”为例的填写样例。

字段结构：

| 字段 | 含义 | 填写要求 |
| --- | --- | --- |
| 课程对象① Who | 学员岗位/角色 | 写清楚谁要学，以及承担的任务 |
| 课程名称② | 课程主题 | 尽量任务化，避免口号化 |
| 课程产生的背景③ Why | 行业、公司、现状三层原因 | 现状角度最重要，要写真实绩效问题 |
| 关键行为（动作）④ What | 课程主题下的关键工作动作 | 3-6 个为宜，动作要能覆盖主题 |
| 透视问题⑤ What | 每个动作背后的学习问题 | 用不想、不知道、不会、做不好表达 |
| 隐藏的内容⑥ How | 课程应覆盖的内容 | 与透视问题一一对应 |
| 涉及的方法⑦ How | 教学方法 | 与内容属性匹配 |

模板注意事项：

1. 关键动作穷尽主题。
2. 写存在的问题，而不是面临的问题；重点看现状角度。
3. 提出基于内容的课程方案，而不是基于管理的解决方案；内容要与问题对应。
